# edufund

Edufund fullstack test App

## Assignment Project for EDUFUND

## STOCK/ETF details display POC App

## Technologies used

1. React for frontend
2. NodeJS/Express for backend
3. Mongodb for database
4. Git for source control

## Initial commit

## Run the app

1. cd frontend/
2. npm i
3. npm start
4. cd backend/
5. npm i
6. npm run migrate
7. npm run migrate:symbol
8. npm start

## To run app with Docker

run `docker-compose up` from root directory
Note: Make sure to uncomment connection line in backend/db/connection.js file and mongodb data is migrated
